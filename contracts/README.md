# LAGOS Governance Contracts

This directory contains the smart contract modules for the LAGOS research framework, focusing on decentralized governance and resource accountability.

## Languages & Paradigms
- **[Move](./move/)**: Resource-oriented paradigm for safe management of overlay assets and bandwidth quotas.
- **[Clarity](./clarity/)**: Decidable smart contracts for predictable, non-terminating policy enforcement.

## Governance Philosophy
LAGOS combines the resource safety of Move with the decidability of Clarity to ensure that all administrative actions within the overlay are both accountable and predictable.
