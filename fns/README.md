# LAGOS Distributed Functions

This directory contains the content-addressed function primitives for the LAGOS framework.

## Languages & Paradigms
- **[Unison](./unison/)**: A language where functions are identified by their content hash, enabling seamless updates and remote execution across the overlay.

## Function Management
By identifying code by its structure rather than its name or version, LAGOS ensures absolute consistency in distributed logic updates, effectively eliminating dependency conflicts.
