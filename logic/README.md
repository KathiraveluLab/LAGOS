# LAGOS Logic & Optimization

This directory contains the real-time optimization and latency analysis modules for the LAGOS framework.

## Languages & Paradigms
- **[Roc](./roc/)**: A functional language compiled to machine code, used for microsecond-scale path selection and packet steering.

## Optimization Strategy
LAGOS leverages the pure functional nature of Roc to implement predictable, high-speed optimization algorithms that minimize tail-latency in federated overlay networks.
